# 1C-проекты

- Привет! Меня зовут Мария Ковеза. Я - программист 1С. 
- В этом репозитории хранятся мои конфигурации 1С:Предприятие 8 в формате файлов поставки (.cf).

## Список конфигураций

| Название файла | Описание |
|----------------|----------|
| Carservice.cf | Конфигурация для автосервиса (учет ремонтов и заказов) |
| CarserviceMP.cf | Мобильное приложение конфигурации для автосервиса |
| KanbanBoard.cf | Конфигурация "Канбан доска" для управления задачами в электороэнергетике|


## Требования к платформе

- Платформа 1С:Предприятие 8.3 (версия не ниже 8.3.10)
- Режим совместимости: Управляемое приложение
